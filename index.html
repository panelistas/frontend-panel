Descripción del proyecto

Este proyecto es un sistema interactivo que permite a los usuarios explorar las constelaciones zodiacales mediante una interfaz táctil.

A través de una tablet, el usuario selecciona una de las doce constelaciones del zodíaco, y el sistema automáticamente dirige un puntero láser hacia la posición correspondiente en el cielo.

Simultáneamente, se proyecta en una página web con información sobre la constelación elegida.
